# Patrones añadidos sin modificar el proyecto existente

La implementación estática se encuentra en `js/patrones.js` y es utilizada por `tienda.html` y `gestion.html`.

| Módulo | Patrones |
|---|---|
| Acceso | Singleton, Proxy, Facade |
| Menú | Mediator, Command, Observer |
| Usuarios | Repository, Factory Method, Memento |
| Clientes | Repository, Prototype, Iterator |
| Proveedores | Repository, Adapter, Bridge |
| Productos | Repository, Builder, Composite, Decorator, Flyweight |
| Ventas | Abstract Factory, Strategy, Template Method, State, Chain of Responsibility |
| Reportes | Interpreter, Iterator, Visitor, Observer, Facade |
